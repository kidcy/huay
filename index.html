📊 ภาพรวมทุกหวย — กดการ์ดเพื่อดูรายละเอียด

เลือกวัน = โชว์เฉพาะหวยที่ออกผล "ตรงวันนั้น" · ตัวที่ไม่ได้ออกจะขึ้นว่า 🔒 ปิด
การ์ดโชว์ 3 ตัวบนของแต่ละหวย · กดเข้าไปดูผลล่าสุด + กางเลข 2 ตัว → 56 (กฎหวยหุ้นเดิม) + ±1 + สถิติ + ประวัติ

📋 ผลหวยทุกหวย ของวันที่เลือก

ตัวที่ยังไม่ออก กรอกผลแล้วกดบันทึกได้เลย · คลิกชื่อหวย 🔎 เพื่อดูเลขแนะนำ + ผลย้อนหลัง

📈 หวยหุ้นดาวโจนส์ — ผลย้อนหลัง (กติกา RUAY)

เลขเปิด = หลักหน่วย+ทศนิยมของราคาเปิด · 3 ตัวบน = หลักหน่วย+ทศนิยมของราคาปิด · 2 ตัวล่าง = ทศนิยมของ Change

🔎 ค้นหาเลขกลับ — กรอกเลข 3 ตัว แตกชุดที่ต้องซื้อ

3 ตัวไม่ซ้ำ → กลับ 6 ตัว · มีเบิ้ล (2 ตัวต่าง) → ครบ 8 ตัว (รวมตอง) · ตอง → 1 ตัว

เครื่องคำนวณสูตร

⚠️ หวยเป็นการสุ่ม สูตรเป็นแนวทาง/ความเชื่อ ไม่รับประกันถูก เล่นพอดีและจำกัดงบเสมอ

วิเคราะห์ "เลขกลับซ้ำ" ทุกหวย

นับงวดที่ 3 ตัวบนเป็นเลขกลับของงวดที่เคยออกแล้ว · "เจอทุกๆ" = เฉลี่ยกี่งวดเจอ 1 ครั้ง

หวยงวดกลับเลขซ้ำเจอทุกๆ(งวด)ระยะห่างเฉลี่ยซ้ำเป๊ะ

🔮 เลขแนะนำ — กลับเลข + สถิติ + สูตรลาว (เฉพาะลาว)

เลือกวันย้อนหลัง = คำนวณเลขแนะนำโดยใช้สถิติ "เฉพาะก่อนวันนั้น" แล้วเทียบกับผลที่ออกจริง (ดูว่ากฎเข้าหรือไม่)

⚠️ สถิติอดีต ไม่ได้ทำนายอนาคต หวยสุ่มล้วน เล่นพอดีและจำกัดงบ

🔁 รอบการออกของกลุ่มเลขกลับ

ประวัติการออกของกลุ่มนี้ (ครั้งที่ 1, 2, 3, ...)

ครั้งที่วันที่เลขที่ออกห่างจากครั้งก่อน(งวด)
⚠️ "อีกกี่งวด" เป็นค่าประมาณจากระยะห่างเฉลี่ยในอดีต ไม่ใช่การทำนายแม่นยำ หวยสุ่มล้วน เล่นพอดี

🇱🇦 สูตรลาว — พีระมิด + วันอ้างอิง

ใช้ 2 วันอ้างอิง: งวดก่อนหน้า + งวดสัปดาห์ก่อน (−7 วัน) · กางพีระมิดหา 2 ตัวล่างสุด → ชุด 3 หลัก 56 ตัว (54 + ตอง 2 ตัว) + เลข ±1

📋 ตารางกางทุกงวด (พีระมิด + ±1)

🔵 น้ำเงิน = เลขออก + กางพีระมิดชิดขวา · 🟢 เขียว = บวก1 / ลบ1 ทุกหลัก · เรียงเก่า → ใหม่
⚠️ สูตรความเชื่อ/แนวทาง · ชุด 56 ตัวกว้างมาก ใช้ดูแนวเฉยๆ หวยสุ่มล้วน ไม่รับประกัน เล่นพอดี

➕ เพิ่มผลงวดใหม่ (บันทึกลงฐานข้อมูลทันที)

ระบบจะถอด 3ตัวบน/2ตัวบน/2ตัวล่าง ให้อัตโนมัติ

🧺 ตะกร้ารวมบิล (เก็บเลขซ้ำได้ · นับบาท)

ใส่เลขซ้ำได้ = ลงหนักเลขนั้น (965 ×2 = 2 บาท ถ้าถูกได้ 2 เท่า) · กดที่เลขในตะกร้าเพื่อลด 1 · ระบบจำในเครื่อง
⚠️ ตะกร้าช่วยรวมยอด/กันลืม ไม่ได้เพิ่มโอกาสถูก หวยสุ่มล้วน เล่นพอดีและจำกัดงบ

🎰 หวยฮานอย — โปรแกรมแยก (สูตรเต็มที่เราทำกัน)

เปิดเต็มหน้าจอ ↗เลขวนปรับตามช่วง · เครื่องคำนวณ ±1 · ชุดนำโชค/ตอง · ตัวจับเลขวน · สถิติ 59 งวด
